Church of Jesus Christ of Latter-day Saints scriptures as a sqlite database

This repo includes a database of all scripture passages used in the standard works of the Church of Jesus Christ of Latter-day Saints.

The ascii files were originally produced by Iron Rod ASCII Scriptures. The author includes a history of how the ascii files were originally obtained and produced here.

To view the database online, I use this sqlite viewer backed by this source code. You just need to point the url to the database stored in this repo with

https://inloop.github.io/sqlite-viewer/?url=https://raw.githubusercontent.com/peterhillyard/lds_scriptures/refs/heads/master/scriptures.db
