Scriptures as a sqlite database

This page has a link to view the scripture passages used in the standard works of the Church of Jesus Christ of Latter-day Saints as a database.

The ascii files were originally produced by Iron Rod ASCII Scriptures. The author includes a history of how the ascii files were originally obtained and produced here. To reiterate, the text is not from official sources.

To view the database online, I use this sqlite viewer backed by this source code. You just need to point the url to the database stored in this repo with

https://inloop.github.io/sqlite-viewer/?url=https://raw.githubusercontent.com/peterhillyard/lds_scriptures/refs/heads/master/scriptures.db
